Overview
--------

The common module provides shared data types, coordinate frame utilities, and dependency injection containers used across all DART-Planner components.

Key Features
------------

- **Data Types**: Standardized data structures for drone state and commands
- **Coordinate Frames**: Coordinate frame transformations and validation
- **Dependency Injection**: Modern DI container for component management
- **Shared Utilities**: Common utilities and constants
